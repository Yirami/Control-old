# Control-old
Air cooling power control  
用于风冷散热的控制器。

---
## 说明
设定多级PWM调速，通过按键手动调节抽风机功率。  
+ `./src-asm/`中为汇编源代码
+ `./src-c/`中为C源代码  
+ `./details/`中为一些细节图片。
